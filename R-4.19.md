# R-4.19

Give a big-Oh characterization, in terms of _n_, of the running time of the `Ex4` function.

    Algorithm Ex4(A):
        Input: An array A storing n ≥ 1 integers.
        Output: The sum of the prefix sums in A.
    s ← A[0]                                             1
    t ← s                                                1
    for i ← 1 to n − 1 do                                n - 1
        s ← s + A[i]                                     1
        t ← t + s                                        1
        { increment i }                                  1
    return t                                             1

f(n) = 1 + 1 + (n - 1)(1 + 1 + 1) + 1
     = 3(n - 1) + 3
     = 3n
     = O(n)
