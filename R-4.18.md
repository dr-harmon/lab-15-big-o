# R-4.18

Give a big-Oh characterization, in terms of _n_, of the running time of the `Ex3` function.

    Algorithm Ex3(A):
        Input: An array A storing n ≥ 1 integers.
        Output: The sum of the prefix sums in A.
    s ← 0                                                   1
    for i ← 0 to n − 1 do                                   n
        s ← s + A[0]                                        1
        for j ← 1 to i do                                   i
            s ← s + A[j]                                    1
            { increment j}                                  1
        { increment i}                                      1
    return s                                                1

f(n) = 1 + n (1 + i(1 + 1) + 1) + 1
     = n(2i + 2) + 2
     = 2ni + 2n + 2
     = 2(n(n + 1)/2) + 2n + 2
     = n(n + 1) + 2n + 2
     = n^2 + 3n + 2
     = O(n^2)
