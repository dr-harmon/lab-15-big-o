# R-4.16

Give a big-Oh characterization, in terms of _n_, of the running time of the `Ex1` function.

    Algorithm Ex1(A):
        Input: An array A storing n ≥ 1 integers.
        Output: The sum of the elements in A.
    s ← A[0]                                          1
    for i ← 1 to n − 1 do                             n
        s ← s + A[i]                                  1
        { increment i }                               1
    return s                                          1

f(n) = 1 + n * (1 + 1) + 1
     = 2n + 2
     = O(n)
